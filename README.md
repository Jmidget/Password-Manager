# Password-Manager
Python code for a GUI that manages username and password information

This program has the following capabilities:
1) Record, website, username, and password info and store it in a file. This file is saved as a JSON
2) Generate a password using random characters (letters, numbers, symbols)
3) Search through JSON data to find saved account information
4) Detects if password information has already been saved nad gives the option to override iwth newer information
